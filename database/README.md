# 🚀 PodMind AI

> The World's Most Powerful AI Platform for Podcasters

PodMind AI is an all-in-one AI-powered SaaS platform designed to help podcasters research, write, optimize, publish, repurpose, and grow their podcasts from a single workspace.

Instead of using 10+ different tools, creators can manage their complete podcast workflow with AI Agents.

---

# 🌍 Vision

Our mission is simple:

> Build the operating system for modern podcasters.

PodMind combines AI research, script writing, SEO, guest discovery, social media automation, knowledge management, analytics, and publishing into one intelligent platform.

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/shafqatgithub/podmind.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Run database migrations

```bash
supabase db push
```
